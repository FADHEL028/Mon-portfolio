SAE 1.03

Installation d’un poste de développement

Contexte

Cette SAÉ consistait à installer un système Linux et à configurer un environnement complet de développement pour la réalisation de projets informatiques.

Objectifs

  • Installer un système Linux
  • Configurer un environnement de développement
  • Mettre en place un serveur web local
  • Installer une base de données

Technologies utilisées

  • Linux (Ubuntu / Kubuntu)
  • Apache2
  • PHP
  • PostgreSQL
  • Java / Python
  • Ligne de commande

Réalisations techniques

  • Création d’une clé USB bootable
  • Installation complète du système
  • Partitionnement du disque
  • Configuration réseau (Eduroam)
  • Installation d’un serveur Apache
  • Déploiement d’un site en local
  • Configuration PostgreSQL
  • Tests avec PHP

Difficultés rencontrées

  • Partitionnement du disque
  • Commandes Linux
  • Configuration des services
  • Gestion des permissions

Résultats

Un environnement de développement complet et fonctionnel a été mis en place, permettant de développer et tester des applications web en local.

Compétences développées

Linux
Administration système
Serveur web
Base de données
Terminal

Exemple

Mise en place d’une application web locale accessible via localhost avec interaction base de données et scripts PHP.
