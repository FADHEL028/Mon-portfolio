SAE 1.01

Développement d’un algorithme de répartition

Contexte

Cette SAÉ consiste à concevoir un programme capable de répartir automatiquement des élèves dans différentes catégories à partir d’un questionnaire. Le projet s’inspire d’un système de décision basé sur des critères définis.

Objectifs

  • Respecter un cahier des charges
  • Développer un programme structuré
  • Implémenter une logique de décision
  • Tester et valider les résultats

Principe de fonctionnement

Le programme repose sur un questionnaire à 4 questions. Chaque réponse est notée entre 1 et 10. L’élève est ensuite affecté à une catégorie en fonction de son score le plus élevé.

  • Question 1 → Profil orienté action
  • Question 2 → Profil intellectuel
  • Question 3 → Profil social
  • Question 4 → Profil ambitieux

Technologies utilisées

  • Python
  • Structures conditionnelles
  • Fonctions
  • Tests unitaires

Méthodologie

  • Analyse du problème
  • Conception de l’algorithme
  • Implémentation en Python
  • Tests et validation

Difficultés rencontrées

  • Gestion des cas d’égalité
  • Structuration du code
  • Mise en place des tests

Résultats

Un programme fonctionnel a été développé permettant de traiter les réponses et de produire une décision cohérente automatiquement.
